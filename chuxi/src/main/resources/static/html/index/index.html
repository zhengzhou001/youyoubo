网站开发

互联网时代任何企业甚至很多个人都需要网站

1：公司网站是公司的形象，网站做的漂亮、访问快、页面简洁大方、产品亮点突出这些特性容易吸引更多客户，相反公司网站做的不好、卡顿、访问慢、经常出错等问题会造成用户的流失。

2：公司后台网站可以帮助企业做一些日常工作，比如用户管理、订单管理、员工管理、考勤管理等企业工作中必然会涉及的工作都可以提需求做网站实现。具有数据方便查询、不会丢失、去纸质化办公等便捷性。

公众号开发

1：微信有高达10亿活跃用户量，由于关注的便捷性（并不需要在手机上另外安装应用软件）导致公众号可以促使公司业务爆发式增长。

2：公众号可以发布文章，只要这些文章写的好，就能提高公司客户的粘性，提供公司活跃用户指数。

3：公众号可以开发公司自己的业务，比如购买公司产品、订单查询、信息录入/修改、机器人客服等等业务。

4：公众号可以设立人工客服，通过于用户聊天解决用户实际问题。

5：公众号关注粉丝数量达到一定级别，可以有广告收入。

6：公众号粉丝最终引流到公司自己网站或者自己App上。
